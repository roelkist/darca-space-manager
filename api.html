API Reference

space_manager.py

A local storage abstraction layer for managing logical spaces. Supports space creation, deletion, listing, and metadata tracking.

This implementation uses the local filesystem and YAML metadata files.

space_file_manager.py

Provides file-level operations within managed logical spaces. Supports reading, writing, deleting, and listing files within spaces, with automatic handling of YAML/JSON content types.
